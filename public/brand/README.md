# Indegenius logo asset pack

Generated from the provided raster/logo image and the three separated logo variants.

## Included
- Transparent PNGs for:
  - Full lockup: icon + wordmark + tagline
  - Icon + wordmark, no tagline
  - Icon only
- White/light transparent PNGs for dark surfaces
- SVG exports for color and white versions
- Simplified icon for favicon/small-size use
- Proper square app/PWA/social icon crops with centered padding
- Monochrome icon: solid black and solid white
- `favicon.ico` plus favicon PNG sizes

## Important SVG note
These SVGs are auto-traced vector paths from the raster artwork, not a native design-tool master export. They are usable as path-based SVG files and do not embed PNG images, but a hand-built/brand-master SVG from Illustrator/Figma would be cleaner for production brand systems.
